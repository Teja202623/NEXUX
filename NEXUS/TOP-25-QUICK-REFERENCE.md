## How to Add to NEXUS Database

### Step 1: Create Database Entries

```sql
INSERT INTO Competitor (
  name,
  website,
  domain,
  facebookPageId,
  instagramHandle,
  industry,
  trades,
  status
) VALUES
(
  'HigherVisibility',
  'https://www.highervisibility.com',
  'highervisibility.com',
  '[FACEBOOK_ID]',
  '@highervisibility',
  'digital-marketing',
  ARRAY['SEO', 'Local SEO', 'Enterprise'],
  'active'
);
```

### Step 2: Get Facebook Page IDs

For each agency, visit their Facebook page and extract ID from URL:
- `facebook.com/[PAGE_ID]`
- Or use Meta Graph API to search by name

### Step 3: Verify Instagram Handles

Visit each agency's Instagram profile and confirm handle:
- Instagram.com/[HANDLE]

### Step 4: Start Scraping

Once database is populated:
```bash
# Get API credentials (see API-SETUP-GUIDE.md)

# Run scraper
npm run scraper:test

# Verify data collection
npm run db:studio
```
